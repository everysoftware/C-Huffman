# C_Huffman
Простой архиватор, сжимающий файлы с помощью алгоритма Хаффмана.  

## Требования
Visual Studio 2022+

## Фичи
* User-friendly UI
* Мета данные хранятся прямо в сжатом файле
* Сжатие файлов до 2 ГБ
